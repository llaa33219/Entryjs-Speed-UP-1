📋 사용 방법

OnlyRun 모드는 Entry.org의 프로젝트를 실행 전용 모드로 볼 수 있는 기능입니다.
URL 형식: http://localhost:8080/onlyrun/{프로젝트ID}

🎮 테스트 링크들

아래는 Entry.org의 실제 프로젝트들을 OnlyRun 모드로 실행하는 링크들입니다:
🎯 테스트 프로젝트 1 🎮 테스트 프로젝트 2 🌟 테스트 프로젝트 3

🔧 구현된 기능들

  • URL 기반 프로젝트 로드: /onlyrun/{프로젝트ID} 형식으로 접근
  • Entry.org GraphQL API 연동: 실시간으로 프로젝트 데이터 가져오기
  • 이미지 경로 자동 변환: 파일명을 절대 링크로 변환
  • 실행 전용 UI: 블록 조립 없이 실행 화면만 표시
  • 자동 실행: 프로젝트 로드 후 자동으로 실행 시작
  • 컨트롤 버튼: 실행/정지, 전체화면 기능
  • 키보드 단축키: 스페이스바(실행/정지), ESC(정지), F11(전체화면)
  • 극강 속도 모드: 기본적으로 활성화된 최고 속도 실행

⚡ 속도 최적화

OnlyRun 모드에서는 기본적으로 극강 속도 모드가 활성화되어 있어 블록들이 최대한 빠르게 실행됩니다:
  • 모든 지연 시간 제거 (wait 블록 등)
  • 하드웨어 블록 지연 최소화
  • 렌더링 최적화
  • 500 FPS로 실행
⚠️ 주의사항:
• 프로젝트 ID가 존재하지 않거나 비공개 프로젝트인 경우 오류가 발생할 수 있습니다.
• 네트워크 연결이 필요합니다 (Entry.org API 호출).
• 일부 하드웨어 관련 블록은 실제 하드웨어 없이는 정상 작동하지 않을 수 있습니다.

🛠️ 개발자 정보

OnlyRun 모드 관련 파일들:
  • example/onlyrun.ejs - OnlyRun 전용 HTML 템플릿
  • webpack_config/serve.js - 웹팩 설정 (라우팅 추가)
